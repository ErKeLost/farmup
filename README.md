<p style="text-align: center;">easy run your TypeScript or JavaScript, powered by <a href="https://github.com/farm-fe/farm">Farm</a></p>

<hr />

feature

- cross import CommonJs and EsModule
- watch mode

## Setup

```bash
pnpm i farmup
```

or install global

```bash
pnpm i -g farmup
```

## Get Started

build

```bash
npx farmup index.ts
```

watch mode

```bash
npx farmup start index.ts
```
